- Bootstrap v5.2.3

- jQuery UI - v1.13.2
- Chart.js v4.4.2

## 功能簡介

此專案為遊戲介紹網站
內容大致為

1.網站首頁
- 網頁訪問動畫(OP)
- 主視覺頁面
- banner(music bar)
- carousel
- timeline
- 說明圖文排版

2.會員功能
- 會員操作介面
- 會員管理
- 商品管理
- 地圖查詢(以opendata數據呈現)
- 圖表統計(以opendata數據呈現)
